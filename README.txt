Atticus Museum Exhibit Site (Offline + Embeddable)

Files:
- index.html  (open this in a browser)
- style.css   (museum styling)
- script.js   (slider controls + animations)
- slides_backup_outline.txt (Google Slides backup)
- image_captions.txt (captions to use under images)
- intro_video_script.txt (30-second script)
- citations_mla_lite.txt (teacher-friendly citations)

How to use:
1) Double-click index.html to run locally.
2) To embed in Google Sites: host the folder on GitHub Pages, Glitch, or CodePen,
   then embed the URL in Google Sites (Insert > Embed).


Images:
- Put optimized images in /images (see images/PUT_IMAGES_HERE.txt).
- The page uses responsive srcset + lazy loading automatically.
